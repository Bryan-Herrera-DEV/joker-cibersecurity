# Joker
Menú básico de funciones de ciberseguridad desarrollado en python3

## Instalación
```bash
git clone https://github.com/Jeremymr2/joker-cibersecurity
cd joker-cibersecurity
pip install -r requirements.txt
```

## Uso
```bash
python joker.py -h
```
